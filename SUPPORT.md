# Support

Use GitHub Issues for reproducible bugs and feature requests in this maintained
fork. Include the exact release tag, CS2/CounterStrikeSharp/.NET versions,
installation layout, configuration with secrets removed, and a minimal
reproduction.

For upstream behavior that is unchanged by this fork, link the corresponding
upstream issue when one exists. Operational incidents affecting the hosted
MBSifu platform should be reported through MBSifu support rather than a public
plugin issue.
